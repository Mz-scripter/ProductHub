Endpoints

  • GET /products - Retrieve products with an optional limit (default: 10, max:100).
  • GET /products/categories - View all available categories.
  • GET /products/search?query= - Search for products by keyword
  • GET /products/{category} - Retrieve products by category with an optional limit (default: 10, max:100)
  • GET /products/{category}/random - Retrieve a random product from a specified category.
  • GET /products/{product_id} - Retrieve a product by its ID.

Examples

Fetch Products in Python

        import requests

        response = requests.get('https://ancient-marti-mz-scripter-730e4701.koyeb.app/products?limit=20')
        print(response.json())

Fetch Products in Node.js

        const axios = require('axios');

        axios.get('https://ancient-marti-mz-scripter-730e4701.koyeb.app/products?limit=20')
            .then(response => console.log(response.data))
            .catch(error => console.error(error));

Error Handling

If a wrong category is used, the API will respond with:

        {
            "error": "Category 'invalid_category' not found. Use /products/categories to view all available categories."
        }
